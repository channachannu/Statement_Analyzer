# ðŸ’° Statement_Analyzer
A modular Streamlit app that analyzes bank statements by salary cycles,  summarizes inflow/outflow, and visualizes spending behavior.


### Features
- Upload CSV/XLSX bank statement
- Identify salary cycles automatically
- Categorize large/small debits
- Calculate spending ratio & cashflow summary
- Visualize monthly trends

### Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
